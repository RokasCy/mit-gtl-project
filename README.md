# The Objective

### Build an autonomous parking assistant

* The duckbot will scan an area until it detects a road (black tape) and begins to follow it.
* It will continue to do so until a parking area (marked with red tape) is detected, where it stops and parks.

**Version 0**

* Parking spot detection
* Scanning area

**Version 1**

* Path follower
* LEDS

**Version 2**

* Extra features
